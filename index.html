02 — Trading & Quant

Quant Projects

⭐ Featured
Gamma-Scalper Bot

A regime-aware options strategy engine targeting SPY. Uses a 6-indicator classification system (RSI, EMA crossover, VWAP, Bollinger Bands %B, VIX, SMA-50) with a consensus voting mechanism — requiring 3+ agreeing signals before triggering any trade. Each detected regime maps to one of 5 options strategies (e.g. trending up → bull call spread, high vol → iron condor).

Built in Python with pandas_ta for indicator computation and Alpaca API for paper trading execution. Previously integrated with IBKR TWS via ib_insync before pivoting to Alpaca after persistent gateway connection issues — a real-world engineering tradeoff that shaped the architecture.
